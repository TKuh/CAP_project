15.2-3 CAP_INTERNAL_CONSTRUCTOR_FOR_TERMINAL_CATEGORY
‣ CAP_INTERNAL_CONSTRUCTOR_FOR_TERMINAL_CATEGORY( options )( function )

Returns: a CAP category

This function takes a record of options suited for CategoryConstructor. It makes common adjustments from TerminalCategory and TerminalCategoryWithMultipleObjects to the list of operations to install and the categorical properties of the given record, before passing it on to CategoryConstructor.
