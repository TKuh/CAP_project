5.1-1 LaTeXStringOfSemisimpleCategoryObjectList
‣ LaTeXStringOfSemisimpleCategoryObjectList( L )( function )

Returns: a string

The argument is a list L defining an object c in a semisimple category. The output is a LaTeX string (without enclosing dollar signs) that may be used to print out c nicely.
